# 🌊 Waves of Food

A native Android food ordering app built with **Kotlin**, featuring Firebase authentication, a clean browsing experience, and a smooth checkout flow.

---

## ✨ Features

- 🔐 **Firebase Authentication** — secure login and sign-up
- 🏠 **Home Feed** — browse curated food items and categories
- 🛒 **Order Flow** — add items to cart and complete purchases
- 🔍 **Search** — quickly find dishes and restaurants
- 👤 **User Profile** — manage account details
- 🎬 **Splash Screen** — polished app launch experience

---

## 🛠️ Tech Stack

| Category | Technology |
|---|---|
| Language | Kotlin |
| Platform | Android (native) |
| Build System | Gradle (Kotlin DSL) |
| Authentication | Firebase Auth |
| Backend / Database | Firebase |

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest stable version recommended)
- A Firebase project with Authentication enabled
- Your own `google-services.json` file added to the `app/` directory

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/SARTHAKHAPPY/Waves_of_Food.git
   ```
2. Open the project in Android Studio
3. Add your `google-services.json` file to the `app/` directory (get this from your Firebase console)
4. Sync Gradle and run the app on an emulator or physical device
